HTML Reference

HTML Tags

Tag Name Description
<h1> to <h6> Heading Defines HTML Headings
<a> Link Tag Defines a hyperlink
<head> Head Tag Defines Head Tag
<body> Body Tag Defines Body Tag
<button> Button Tag Defines a Clickable button

HTML Attributes

Attribute Name Description
src Source Attribute Specifies the path to the image to be displayed
width Width Attribute Defines the width of the image
height Height Attribute Defines the height of the image
style Style Attribute Used to add styles to an element such as color, size, font ,etc.
